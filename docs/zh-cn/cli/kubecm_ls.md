## kubecm ls

列举所有 Context

### 简介

列举所有 Context

```
kubecm ls
```

### 示例

```

# List all the contexts in your kubeconfig file
kubecm ls
# Aliases
kubecm l

```

### 选项

```
  -h, --help   help for ls
```

### 全局选项

```
      --config string   path of kubeconfig (default "/Users/guoxudong/.kube/config")
```
